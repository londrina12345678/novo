Tags HTML

<p> - Parágrafo

O elemento <p> é usado para criar parágrafos de texto.

<img> - Imagem

O elemento <img> é usado para incluir imagens em uma página HTML.

<h1> a <h6> - Títulos

Os elementos <h1> a <h6> são usados para definir títulos de diferentes níveis.

Propriedades CSS

Centralização

Para centralizar um elemento no CSS, você pode usar a propriedade text-align para textos e margin com valor auto para elementos de bloco.

Cor

A propriedade color no CSS define a cor do texto dentro de um elemento.

Tamanho

Para definir o tamanho de um elemento no CSS, você pode usar as propriedades width e height.
